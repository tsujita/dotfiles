Worker::start

(PECL pthreads >= 0.36)

Worker::start — 実行する

説明

final public boolean Worker::start ( void )

新しいスレッドを開始し、Worker::run を実行して Stackable を待ちます。

パラメータ

この関数にはパラメータはありません。

返り値

成功したかどうかを表す boolean 値を返します。
