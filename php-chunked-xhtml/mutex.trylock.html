Mutex::trylock

(PECL pthreads >= 0.34)

Mutex::trylock — ミューテックスの確保を試みる

説明

final public static boolean Mutex::trylock ( long $mutex )

呼び出し元用に、別スレッドが所有する (ロックしている) ミューテックスを、ブロックせずにロックしようと試みます。

パラメータ

mutex

Mutex::create() を呼んで取得したハンドル。

返り値

成功したかどうかを返します。
